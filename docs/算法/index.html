深拷贝​

标签：递归
核心思路：利用 WeakMap/Map 实现（推荐使用 WeakMap，自动垃圾回收）

/**
 * 递归实现深拷贝
 * @param {*} obj 拷贝对象
 * @param {*} hash WeakMap类型：键值对形式，key必须是对象，value任意类型，会自动垃圾回收无用key
 * @returns 
 */
function deepClone(obj, hash = new WeakMap()) {
  if (obj instanceof RegExp) return new RegExp(obj);
  if (obj instanceof Date) return new Date(obj);
  if (obj === null || typeof obj !== 'object') return obj; //直接返回基础类型
  if (hash.has(obj)) return hash.get(obj); //处理循环引用

  let clone = new obj.constructor; //获取目标构造函数 并实例化
  hash.set(obj, clone);

  for (let key in obj) {
    if (Object.hasOwnProperty.call(obj, key)) {
      clone[key] = deepClone(obj[key], hash); //递归
    }
  }

  return clone;
}

动态规划​

爬楼梯 Leetcode:70​

难度：简单 标签：数学 动态规划 记忆化搜索
核心思路：递归也能解。但如果就让你使用动态规划，在没有思路的时候，不妨逆向推导，站在终点，思考回退的可能性。在n的位置我们可以得出关系：f(n) = f(n-1) + f(n-2)，动态规则要求自底向上的过程，根据已知的方程，不难得出 f(3) = f(1) + f(2)，f(1) 和 f(2) 都可知，这样只要一步步得出每层所需的数字并积累，最后算到n，即得解。

/* 【题目】
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。
每次你可以爬 1 或 2 个台阶。你有多少种不同的方法可以爬到楼顶呢？ 
*/

/**
 * 动态规划解法
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
  // 1.定义固定的初始值
  const f = [0, 1, 2];
  // 2.从3开始，到n结束，f[n]代表爬到第n层有多少方法
  for (let i = 3; i <= n; i++) {
    f[i] = f[i - 1] + f[i - 2];
  }
  // 3.返回爬n层所需的方法数量
  return f[n];
}

/**
 * 扩展：递归解法
 * @param {number} n
 * @return {number}
 */
var climbStairs = function(n) {
  // 处理递归边界
  if (n === 1) {
    return 1
  }
  if (n === 2) {
    return 2
  }
  // 递归计算
  return climbStairs(n - 1) + climbStairs(n - 2)
}

零钱兑换 Leetcode:322​

难度：中等
标签：数组 广度优先搜索 动态规划
核心思路：将总金额拆分为一个从0到max的数组，初始化[0]，每次遍历下一个金额时，初始化数据为Infinity(无穷大)，当遍历不同面额的数组时，满足一定条件，都能通过比较大小，将更小的结果覆盖掉初始的无穷大。最终即可得到结果，若结果是Infinity，则也说明无法组成总金额，自然可以返回-1。

/* 【题目】
给你一个整数数组 coins ，表示不同面额的硬币；以及一个整数 amount ，表示总金额。
计算并返回可以凑成总金额所需的 最少的硬币个数 。如果没有任何一种硬币组合能组成总金额，返回-1。
你可以认为每种硬币的数量是无限的。
示例 1：
输入：coins = [1, 2, 5], amount = 11
输出：3 
解释：11 = 5 + 5 + 1

示例 2：
输入：coins = [2], amount = 3
输出：-1

示例 3：
输入：coins = [1], amount = 0
输出：0
*/

/**
 * @param {number[]} coins
 * @param {number} amount
 * @return {number}
 */
var coinChange = function (coins, amount) {
  // 0.声明一个结果集，代表每个金额所需的最少硬币数，初始化0硬币所需数量为0
  const res = [0];

  // 1.第一层循环：从总1个硬币开始，到总金额结束
  for (let i = 1; i <= amount; i++) {
    res[i] = Infinity;
    // 2.第二层循环：遍历不同面额的硬币
    for (let j = 0; j < coins.length; j++) {
      // 判断当前面额的硬币小于等于当前所需硬币数
      if (coins[j] <= i) {
        // 当前值和原来值进行比较，得到当前面额的硬币所需的最少硬币数
        res[i] = Math.min(res[i], res[i - coins[j]] + 1);
      }
    }
  }

  // 检查结果是否为 无穷大(Infinity)，说明没有任何一种硬币组合能组成总金额
  if (res[amount] === Infinity) return -1;

  // 返回结果（组合所需的最少硬币数）
  return res[amount]
};

0-1背包​

难度：简单
标签：数组 动态规划
核心思路：动态规划思想，定义一个总体积的"背包"（体积从0-max），遍历每个物品，将各个体积对应的最大价值记录下来：第一次用第1个物品处理一遍总体积，再用第2个物品基于上一次数据处理当前的总体积，记录更大的价值.... 依次循环 动态迭代数据，即能得出最大体积对应的最大价值。

/* 【题目】
有 n 件物品，物品体积用一个名为 w 的数组存起来，物品的价值用一个名为 value 的数组存起来；
每件物品的体积用 w[i] 来表示，每件物品的价值用 value[i] 来表示。
现在有一个容量为 c 的背包，问你如何选取物品放入背包，才能使得背包内的物品总价值最大？
注意：每种物品都只有1件

示例 1：
输入：2,10,[1,10],[3,4]
输出：4
解释：第一个物品的体积为1，价值为3，第二个物品的体积为10，价值为4。只取第二个物品可以达到最优方案，总价值为4

示例 2：
输入：2,10,[1,9],[3,8]
输出：11
解释：两个物品体积之和等于背包能装的体积，所以两个物品都取是最优方案
*/

/**
 * @param {number} n 物品数量
 * @param {number} c 背包容量
 * @param {number[]} w 物品体积数组
 * @param {number[]} value 物品价值数组
 * @return {number} 背包内物品总价值最大值
 */
var knapsack = function (n, c, w, value) {
  // 0.初始化各个体积对应的总价值
  const dp = new Array(c + 1).fill(0);

  // 1.第一层循环：从第1个物品开始，遍历所有物品
  for (let i = 0; i < n; i++) {
    // 2.第二层循环：从最大容量开始，遍历背包的容量，每次递减1
    for (let j = c; j >= w[i]; j--) {
      // 比较当前体积的总价值和之前记录的总价值，取较大值
      dp[j] = Math.max(dp[j], dp[j - w[i]] + value[i])
    }
  }

  // 返回最大体积对应的结果，即最大价值
  return dp[c];
};

分割等和子集 Leecode:416​

难度：中等
标签：数组 动态规划
核心思路：首先判断是否符合可计算原则，通过前置条件直接排除掉不符合的数据。然后可以利用【01背包】思想解决，或者也推荐此方程思想：dp[j] = dp[j] ∣ dp[j−nums[i]]，利用第二层循环的从大到小计算，直接就能通过当前数字基于他的互补数字判断是否可以组合成该数字的结果(boolean)，最终即得解。

/* 【题目】
给你一个 只包含正整数 的 非空 数组 nums。
请你判断是否可以将这个数组分割成两个子集，使得两个子集的元素和相等。

示例 1：
输入：nums = [1,5,11,5]
输出：true
解释：数组可以分割成 [1, 5, 5] 和 [11]

示例 2：
输入：nums = [1,2,3,5]
输出：false
解释：数组不能分割成两个元素和相等的子集
*/

/**
 * @param {number[]} nums
 * @return {boolean}
 */
var canPartition = function (nums) {
  /** 前置条件（符合直接判定为false）
   * 1.长度不足2
   * 2.总和为奇数
   * 3.最大值大于二分数
   */
  // 1.判断长度是否不足2（false)
  if (nums.length < 2) return false;

  // 2.计算数组中的最大值 和 数组总和
  let maxNum = 0;
  const sum = nums.reduce((a, b) => {
    maxNum = maxNum > b ? maxNum : b;
    return a + b;
  }, 0);

  // 3.计算二分数
  const target = sum / 2;

  // 4.判断总和为奇数||最大值大于二分数（false)
  if (sum & 1 === 1 || maxNum > target) {
    return false;
  }

  /* 【方式一】01背包 */
  // 1.定义一个长度为二分数+1，值都为0的数组
  const dp = new Array(target + 1).fill(0);
  // 2.第一层循环：遍历[正整数]数组
  for (let i = 0; i < nums.length; i++) {
    // 3.第二层循环：从二分数开始递减
    for (let j = target; j >= nums[i]; j--) {
      // 比较当前总和和之前记录的总和，取较大值
      dp[j] = Math.max(dp[j], dp[j - nums[i]] + nums[i]);
      // 判断当前总和是否等于目标值(二分数)（true）
      if (dp[j] === target) return true;
    }
  }
  // 4.无法达到目标值，直接返回false
  return false;
};
/**
 * @param {number[]} nums
 * @return {boolean}
 */
var canPartition = function (nums) {
  // 1.判断长度是否不足2（false)
  if (nums.length < 2) return false;

  // 2.计算数组中的最大值 和 数组总和
  let maxNum = 0;
  const sum = nums.reduce((a, b) => {
    maxNum = maxNum > b ? maxNum : b;
    return a + b;
  }, 0);

  // 3.计算二分数
  const target = sum / 2;

  // 4.判断总和为奇数||最大值大于二分数（false)
  if (sum & 1 === 1 || maxNum > target) {
    return false;
  }

  /* 【方式二】 */
  // 1.定义一个长度为二分数+1，值都为false的数组
  const dp = new Array(target + 1).fill(false);
  // 初始化第零项为true
  dp[0] = true;
  // 2.第一层循环：forof遍历[正整数]数组项
  for (const num of nums) {
    // 3.第二层循环：从二分数开始递减
    for (let j = target; j >= num; --j) {
      // ❗ 当前值和互换值进行按位或运算，存在true时，保持不变
      dp[j] = dp[j] | dp[j - num];
      // 判断目标值(二分数)对应的值是否已满足（true）
      if (dp[target]) return true;
    }
  }
  // 4.无法达到目标值，直接返回false
  return false;
};

一和零 Leetcode:474​

难度：中等
标签：数组 字符串 动态规划
核心思路：暂略

/* 【题目】
给你一个二进制字符串数组 strs 和两个整数 m 和 n。
请你找出并返回 strs 的最大子集的长度，该子集中 最多 有 m 个 0 和 n 个 1。
如果 x 的所有元素也是 y 的元素，集合 x 是集合 y 的 子集。

示例 1：
输入：strs = ["10", "0001", "111001", "1", "0"], m = 5, n = 3
输出：4
解释：最多有 5 个 0 和 3 个 1 的最大子集是 {"10","0001","1","0"} ，因此答案是 4。
其他满足题意但较小的子集包括 {"0001","1"} 和 {"10","1","0"} 。{"111001"} 不满足题意，因为它含 4 个 1 ，大于 n 的值 3。

示例 2：
输入：strs = ["10", "0", "1"], m = 1, n = 1
输出：2
解释：最大的子集是 {"0", "1"} ，所以答案是 2。
*/

/**
 * @param {string[]} strs
 * @param {number} m
 * @param {number} n
 * @return {number}
 */
var findMaxForm = function (strs, m, n) {
  // let res = 0; ❌错误：
  /**
   * 误区：此子集是指集合中所有元素汇总的_m和_n不超过m和n，而不是集合中的单项元素不超过m和n
   * 此题不是多重背包，是属于01背包，但有两种维度（m和n）
   */

  // 1.初始化dp数组，m和n组成的二维数组 [[0,0],[0,0]...]
  const dp = Array.from(new Array(m + 1), () => new Array(n + 1).fill(0));
  // 2.遍历strs数组的每项元素
  for (const str of strs) {
    // 3.初始化 _m 和 _n 对应的个数为0
    let _m = 0, _n = 0
    // 4.遍历strs中的当前项字符串str
    for (const s of str) {
      // 累计_m（0的个数）和_n（1的个数）
      if (s === "0") _m++;
      else if (s === "1") _n++;
    }

    // 5.第一层循环：倒叙遍历m（0的个数）
    for (let i = m; i >= _m; i--) {
      // 6.第二层循环：倒叙遍历n（1的个数）
      for (let j = n; j >= _n; j--) {
        // 比较之前值和当前值（去掉_m和_n对应的背包个数+当前元素个数1），取较大值
        dp[i][j] = Math.max(dp[i][j], dp[i - _m][j - _n] + 1)
      }
    }

  }

  // 返回最终结果
  return dp[m][n];
};

目标和 Leetcode:494​

难度：中等
标签：数组 动态规划 回溯
核心思路：本题是求组合数，非以往的最值，核心难点在于递推公式的推理，f(n)=f(n-1)+fn(n-2)+...+f(0)

/* 【题目】
给你一个非负整数数组 nums 和一个整数 target 。
向数组中的每个整数前添加 '+' 或 '-' ，然后串联起所有整数，可以构造一个 表达式 ：
例如，nums = [2, 1] ，可以在 2 之前添加 '+' ，在 1 之前添加 '-' ，然后串联起来得到表达式 "+2-1" 。
返回可以通过上述方法构造的、运算结果等于 target 的不同 表达式 的数目。

示例 1：
输入：nums = [1,1,1,1,1], target = 39m
输出：5
解释：一共有 5 种方法让最终目标和为 3 。
-1 + 1 + 1 + 1 + 1 = 3
+1 - 1 + 1 + 1 + 1 = 3
+1 + 1 - 1 + 1 + 1 = 3
+1 + 1 + 1 - 1 + 1 = 3
+1 + 1 + 1 + 1 - 1 = 3

示例 2：
输入：nums = [1], target = 1
输出：1
*/

/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number}
 */
var findTargetSumWays = function (nums, target) {
  /**
   * ∵left+right = sum
   * ∵left-right = target
   * ∴left = (sum+target)/2
   */
  const sum = nums.reduce((a, b) => a + b, 0);
  const left = (sum + target) / 2;

  /* 直接判定为0，无需计算
     1.计算目标前为奇数
     2.目标和大于集合总和
  */
  if ((sum + target) & 1 === 1 || Math.abs(target) > sum) return 0;

  // 1.dp初始化
  const dp = new Array(left + 1).fill(0);
  // 下标0初始化为：1
  dp[0] = 1;

  // 2.确定遍历顺序
  // 第一层循环：遍历nums数组
  for (let i = 0; i < nums.length; i++) {
    // 第二层循环：倒叙遍历正数集合
    for (let j = left; j >= nums[i]; j--) {
      // 3.递推公式：dp[j]=dp[j-1]+dp[j-2]+...dp[0]
      dp[j] += dp[j - nums[i]];
    }
  }

  return dp[left];
};

零钱兑换II Leetcode:518​

/* 【题目】

示例 1：

解释：有四种方式可以凑成总金额：

示例 2：
输入：amount = 3, coins = [2]

示例 3：

/**
 * @param {number} amount
 * @param {number[]} coins
 * @return {number}
 */
var change = function (amount, coins) {
  // 1.初始化dp数组
  const dp = new Array(amount + 1).fill(0);
  // 下标0赋值为 1
  dp[0] = 1;

  /* 计算的是排列组合数 ❌ */
  // for (let i = 1; i <= amount; i++) {  //遍历背包容量
  //   for (let j = 0; j <= coins.length; j++) {  //遍历物品
  //     if (i >= coins[j]) dp[i] += dp[i - coins[j]]
  //   }
  // }

  /* 计算的是组合数 ✅ */
  // 2.第一层循环：遍历硬币集合（遍历物品）
  for (let i = 0; i <= coins.length; i++) {

    for (let j = coins[i]; j <= amount; j++) {

      dp[j] += dp[j - coins[i]]
    }
  }

  // 返回最终结果
  return dp[amount];
};

组合总和Ⅳ Leetcode:377​

难度：中等
标签：数组 动态规划
核心思路：完全背包问题，正序遍历背包容量， 此题求解的是排列组合数，合理利用循环选取元素的顺序，先遍历背包容量，再遍历物品集合，再结合递归公式，所解即得排列组合数。

/* 【题目】
给你一个由 不同 整数组成的数组 nums ，和一个目标整数 target 。请你从 nums 中找出并返回总和为 target 的元素组合的个数。
题目数据保证答案符合 32 位整数范围。
 
示例 1：
输入：nums = [1,2,3], target = 4
输出：7
解释：
所有可能的组合为：
(1, 1, 1, 1)
(1, 1, 2)
(1, 2, 1)
(1, 3)
(2, 1, 1)
(2, 2)
(3, 1)
请注意，顺序不同的序列被视作不同的组合。

示例 2：
输入：nums = [9], target = 3
输出：0
*/

/**
 * @param {number[]} nums
 * @param {number} target
 * @return {number}
 */
var combinationSum4 = function (nums, target) {
  // 1.初始化dp数组
  const dp = new Array(target + 1).fill(0);
  // 下标0赋值为 1
  dp[0] = 1;

  // 2.第一层循环：遍历目标数（遍历背包容量）
  for (let i = 1; i <= target; i++) {
    // 3.第二层循环：遍历数字集合（遍历物品）
    for (let j = 0; j < nums.length; j++) {
      // 利用递推公式，求解背包中不同容量对应的排列组合数
      if (i >= nums[j]) dp[i] += dp[i - nums[j]];
    }
  }

  // 返回最终结果
  return dp[target];
};

爬楼梯（进阶）​

标签：数组 动态规划
核心思路：1阶，2阶，.... m阶是物品，楼顶 n阶是背包，可利用完全背包求解，还是排列组合的问题，双重循环并判断条件，符合后使用递推公式即可求解。

/* 【题目】
假设你正在爬楼梯。需要 n 阶你才能到达楼顶。 
每次你可以爬至多m (1 <= m < n)个台阶。你有多少种不同的方法可以爬到楼顶呢？ 
注意：给定 n 是一个正整数。

示例 1：
输入：n = 3，m = 2
输出：3
解释：当 m = 2，n = 3 时，n = 3 这表示一共有三个台阶，m = 2 代表你每次可以爬一个台阶或者两个台阶。
此时你有三种方法可以爬到楼顶。
1 阶 + 1 阶 + 1 阶段
1 阶 + 2 阶
2 阶 + 1 阶
*/

/**
 * @param {number} n
 * @param {number} m
 * @return {number}
 */
var climbStairs = function (n, m) {
  const dp = new Array(n + 1).fill(0);
  dp[0] = 1;
  for (let i = 1; i <= n; i++) {
    for (let j = 1; j <= m; j++) {
      // 符合条件后，使用递推公式
      if (i >= j) dp[i] += dp[i - j];
    }
  }
  return dp[n]
};

零钱兑换 Leetcode:322​

难度：中等
标签：数组 动态规划 广度优先搜索
核心思路：完全背包问题，正序遍历，先遍历物品或者容量都可以，推荐先遍历物品，减少循环次数以及无需手动判断。然后利用最值的递推公式，求出 Min 的结果。

/* 【题目】
给你一个整数数组 coins ，表示不同面额的硬币；以及一个整数 amount ，表示总金额。
计算并返回可以凑成总金额所需的 最少的硬币个数 。如果没有任何一种硬币组合能组成总金额，返回 -1 。
你可以认为每种硬币的数量是无限的。

示例 1：
输入：coins = [1, 2, 5], amount = 11
输出：3 
解释：11 = 5 + 5 + 1

示例 2：
输入：coins = [2], amount = 3
输出：-1

示例 3：
输入：coins = [1], amount = 0
输出：0
*/

/**
 * @param {number[]} coins
 * @param {number} amount
 * @return {number}
 */
var coinChange = function (coins, amount) {
  const dp = new Array(amount + 1).fill(Infinity);
  dp[0] = 0;

  for (let i = 0; i < coins.length; i++) {
    for (let j = coins[i]; j <= amount; j++) {
      dp[j] = Math.min(dp[j], dp[j - coins[i]] + 1)
    }
  }

  return dp[amount] === Infinity ? -1 : dp[amount];
};

完全平方数 Leetcode:279​

难度：中等
标签：数组 动态规划 广度优先搜索
核心思路：完全背包问题，n是背包容量，从1开始遍历，每次计算的平方数就是当前的物品，同时物品满足 <= 最大容量，即可轻松利用背包问题求解。

/* 【题目】
给你一个整数 n ，返回 和为 n 的完全平方数的最少数量 。
完全平方数 是一个整数，其值等于另一个整数的平方；换句话说，其值等于一个整数自乘的积。例如，1、4、9 和 16 都是完全平方数，而 3 和 11 不是。

示例 1：
输入：n = 12
输出：3 
解释：12 = 4 + 4 + 4

示例 2：
输入：n = 13
输出：2
解释：13 = 4 + 9
*/

/**
 * @param {number} n
 * @return {number}
 */
var numSquares = function (n) {
  // 1.初始化dp数组
  const dp = new Array(n + 1).fill(Infinity);
  dp[0] = 0;

  // 2.第一层循环：从1开始遍历，并判断平方数<=n
  for (let i = 1; i * i <= n; i++) {
    // 计算平方数
    const value = i * i; //⚠也可使用 i**2 求平方数，但**的效率较低，不推荐使用
    // 3.第二层循环：从平方数开始，遍历到n
    for (let j = value; j <= n; j++) {
      dp[j] = Math.min(dp[j], dp[j - value] + 1)
    }
  }

  return dp[n];
};

最长递增子序列 Leetcode:300​

难度：中等
标签：数组 动态规划 二分查找
核心思路：掌握动规顺序、递推公式，获取每个数组从 0 到 i 对应的最长子序列的长度。

/* 【题目】
给你一个整数数组 nums ，找到其中最长严格递增子序列的长度。
子序列 是由数组派生而来的序列，删除（或不删除）数组中的元素而不改变其余元素的顺序。例如，[3,6,2,7] 是数组 [0,3,1,6,2,2,7] 的子序列。

示例 1：
输入：nums = [10,9,2,5,3,7,101,18]
输出：4
解释：最长递增子序列是 [2,3,7,101]，因此长度为 4 。

示例 2：
输入：nums = [0,1,0,3,2,3]
输出：4

示例 3：
输入：nums = [7,7,7,7,7,7,7]
输出：1
*/

/**
 * @param {number[]} nums
 * @return {number}
 */
var lengthOfLIS = function (nums) {
  // 1.初始化dp数组
  const dp = new Array(nums.length).fill(1);
  // 定义结果变量
  let result = 1;

  // 2.第一层循环：从 1 开始遍历 nums
  for (let i = 1; i < nums.length; i++) {
    // 3.第二层循环：从 0 开始遍历到 i
    for (let j = 0; j < i; j++) {
      // 判断当前数字>之前数字
      if (nums[i] > nums[j]) {
        // 4.利用递推公式设置当前i对应的最长递增子序列的长度
        dp[i] = Math.max(dp[j] + 1, dp[i]);
        // 保留最大值
        dp[i] > result && (result = dp[i]);
      }
    }
  }

  // 返回最终结果
  return result;
};

最长连续递增序列 Leetcode:674​

难度：简单
标签：数组
核心思路：局部最优(局部递推) → 全局最优解(动态规划出结果)。前后2个值相比较，令连续长度+1。

/* 【题目】
给定一个未经排序的整数数组，找到最长且 连续递增的子序列，并返回该序列的长度。
连续递增的子序列 可以由两个下标 l 和 r（l < r）确定，如果对于每个 l <= i < r，都有 nums[i] < nums[i + 1] ，
那么子序列 [nums[l], nums[l + 1], ..., nums[r - 1], nums[r]] 就是连续递增子序列。

示例 1：
输入：nums = [1,3,5,4,7]
输出：3
解释：最长连续递增序列是 [1,3,5], 长度为3。
尽管 [1,3,5,7] 也是升序的子序列, 但它不是连续的，因为 5 和 7 在原数组里被 4 隔开。 

示例 2：
输入：nums = [2,2,2,2,2]
输出：1
解释：最长连续递增序列是 [2], 长度为1。
*/

/**
 * @param {number[]} nums
 * @return {number}
 */
var findLengthOfLCIS = function (nums) {
  // 1.初始化dp数组
  const dp = new Array(nums.length).fill(1);
  // 初始化结果
  let result = 1;

  // 2.一层循环：遍历数组nums
  for (let i = 0; i < nums.length; i++) {
    // 3.判断下一个元素是否大于当前元素，是则更新dp数组
    if (nums[i + 1] > nums[i]) {
      // dp数量为上一个元素对应的数据+1
      dp[i + 1] = dp[i] + 1;
      // 保留最大值
      result = Math.max(result, dp[i] + 1);
    }
  }

  // 返回最终结果
  return result;
};

/**
 * 贪心方案
 * @param {number[]} nums
 * @return {number}
 */
const findLengthOfLCIS = (nums) => {
  // 1.定义三个容器：结果、当前最大值、当前数值
  let result = 1, curMax = 1, curNum = nums[0];

  // 2.遍历nums
  for (let num of nums) {
    // 3.判断遍历的数值是否大于当前数值
    if (num > curNum) {
      // 当前最大值+1
      curMax++;
      // 结果赋值较大值（结果和当前最大值）
      result = Math.max(result, curMax);
    } else {
      // 不满足判断条件，当前最大值重置为1
      curMax = 1;
    }
    // 当前数值赋值为遍历的数值
    curNum = num;
  }

  // 返回最终结果
  return result;
};
