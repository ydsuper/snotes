2023 年​

  • 12 月 01 日（Fri）

    技术中心平台：

    • echarts 图表的 legend 调整到左边，toolbox 调整位置等，默认启用 takeGlobalCursor 功能
    • dataZoom 组件，给所有图表添加 slider 类型的区域缩放组件
    • 调整图表下方的数据标签排版

  • 12 月 04 日（Mon）

    技术中心平台：

    • echarts 图表增加中键操作，取代鼠标左键长按拖动图表（ps: 4⭐）
    • 调整了 takeGlobalCursor 功能的开启方式，之前是默认开启，改为手动按下左键开启，松开左键手动关闭。这种方式，解决了其和自定义的中键功能产生冲突的问题
    • 优化了 echarts 图表的 dataZoom 事件处理，维护折线图的 dataZoom 百分比数据

  • 12 月 05 日（Tues）

    技术中心平台：

    • echarts 图表添加 markArea 时，隐藏所有标记名，鼠标悬浮自动显示标记名，同时解决在 Tabs 切换图表时，自动同步的 markArea 无法显示标记名问题，手动在初始化阶段延迟再次执行 setOption 重绘图表
    • 维护 dataZoom 百分比数据，拖动 dataZoom:slider 横轴组件时未维护

  • 12 月 06 日（Wed）

    技术中心平台：

    • echarts 下方数据添加美术场景 tag
    • 根据后端数据，调整节点树构建，绘制 markLine 时更新节点和美术场景数据(tag 展示)

    YDSNotes：

    • 搭建初版学习笔记，配置 workflows，尝试 Github Action 自动部署

    • YDSNotes 成功自动化部署到 Github Pages，采用 Remote 方式，from private to public

      配置后尝试部署失败 2 次，复盘：

      • 使用 yarn 命令进行部署，不能忽略上传 yarn.lock 文件
      • 想当然的以为这个项目 Action 需要用到的私钥(secrets TOKEN)，和之前配置博客的一样，导致部署失败，查看官方文档，实际需要的是 SSH key

  • 12 月 07 日（Thur）
